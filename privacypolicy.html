TabSniffer Privacy Policy

Effective date: March 7, 2026

This privacy policy describes how TabSniffer handles data when you use the extension.

1. What the extension does

TabSniffer captures and displays network activity for the browser tab you launch it from, including request lifecycle events, payload summaries, and headers when available.

2. Data handling model

  • Captured request records are handled in extension runtime memory.
  • Records are scoped per tab for the active extension session.
  • The extension provides tools to clear records and export visible rows to CSV.

3. Data we do not collect or sell

  • We do not sell personal information.
  • We do not require account sign-in.
  • We do not intentionally transmit captured logs to a remote backend service.

4. Permissions and why they are needed

  • webRequest: Observe request lifecycle events and metadata.
  • <all_urls> host permission: Required to observe requests across sites when you use the extension.

5. User controls

  • Pause/Play capture for a tab.
  • Clear records for a tab.
  • Reset filters/search fields.
  • Export visible table rows to CSV.

6. Data retention and control

You can clear displayed records directly from the UI. Closing tabs or removing the extension may clear extension-managed runtime data according to browser behavior.

7. Policy alignment

This policy is intended to align with Chrome Web Store transparency and disclosure expectations.
